About This Project

This EEG Upload System allows researchers to manage EEG data collection workflows. It connects a Flask backend with EEG hardware registration, metadata uploads, and automatic EEG file processing.

It was developed by Aditya to simplify the integration between EEG devices and data preprocessing pipelines.
